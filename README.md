# Sure automation task

Automation framework for OrangeHRM technical assessment  
Built with **Selenium**, **TestNG**, and **RestAssured** following the **Page Object Model (POM)** pattern.

## Features
- UI automation (Selenium WebDriver)
- API automation (RestAssured)
- TestNG-based test runner
- Config-driven setup

## How to Run
mvn clean test
